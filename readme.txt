Домашнее задание No 10.2
Написать программу – список задач.
Каждая задача имеет:
■ заголовок;
■ описание;
■ приоритет;
■ дату создания;
■ дату окончания;
■ кто будет выполнять;
■ отметка о выполнении.
Добавленные задачи должны быть сохранены в файл и
считаны при повторной загрузке программы.
